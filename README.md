# Pravin Gyawali — Developer Portfolio

A modern, interactive developer portfolio built with React, focused on showcasing software engineering skills, projects, cybersecurity interests, certifications, and continuous technical growth.

The goal of this project is to create a premium developer experience combining:

- Modern frontend engineering
- Clean component architecture
- Responsive UI development
- Interactive animations
- Security-focused engineering principles
- Scalable React patterns

---

This portfolio is designed to be more than a personal website.

# 🏗️ Architecture

The application follows a scalable React component architecture:

```

 |
 └── MainLayout
        |
        ├── Navbar
        |
        ├── Home Page
        |
        │    ├── Hero
        │    ├── About
        │    ├── Projects
        │    ├── Skills
```
